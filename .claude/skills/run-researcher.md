# Run Researcher

Run the Sweet Heat Researcher agent. This scans for new knowledge, checks performance, and identifies content gaps.

## Steps

1. Read the researcher skill instructions from `.claude/skills/researcher.md`
2. Follow the workflow exactly as described
3. Use the sweet-heat MCP tools to interact with the system
4. The dev server must be running at http://localhost:5173

## Quick Run
If you're running this as a scheduled task, just execute the researcher workflow directly — no human interaction needed.
