# Run Strategist

Run the Sweet Heat Strategist agent. This generates strategic content seeds for all active brands.

## Steps

1. Read the strategist skill instructions from `.claude/skills/strategist.md`
2. Follow the workflow exactly as described
3. Use the sweet-heat MCP tools to interact with the system
4. The dev server must be running at http://localhost:5173

## Quick Run
If you're running this as a scheduled task, just execute the strategist workflow directly — no human interaction needed.
